# Skills

Repositório de skills personalizadas para agentes de IA.

## Instalação

Para instalar as skills deste repositório, utilize a CLI oficial [skills.sh](https://skills.sh):

```bash
npx skills add viniciuscsouza/skills
```

Isso fará com que o `skills` procure as skills dentro deste repositório e as instale no seu ambiente.

## Estrutura do Repositório

- `ai-driven/`: Conjunto de skills focadas em fluxos de trabalho guiados por IA.
